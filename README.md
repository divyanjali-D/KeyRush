# KeyRush ⚡

KeyRush is a fast-paced, interactive application designed to test, challenge, and improve your keyboard typing speed and reflexes. Built with a clean, modern user interface, KeyRush tracks performance metrics in real-time to help users master their keystrokes and accelerate their typing efficiency.

---

## 🚀 Features

- **Real-Time Analytics:** Live tracking of Words Per Minute (WPM), Keystrokes Per Minute (KPM), and Accuracy percentage.
- **Dynamic Difficulty Modes:** Choose from various difficulty settings (Easy, Medium, Hard) or custom time limits (15s, 30s, 60s).
- **Comprehensive Score Summary:** Offers a deep-dive breakdown of performance at the end of each session.
